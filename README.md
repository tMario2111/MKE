# MKE

## Description

MKE in a simple extension to SFML that comes with the following features:
- State machine
- Asset manager
- Sound manager
- Animations 
- Delta time
- Input
- Random 
- Shadow caster
- Sprite batching 
- Particle manager
- Other utilities
